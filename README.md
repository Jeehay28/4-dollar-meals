## 💡 Introduction
### 1. 프로젝트 개요
- 제작기간 : 2022. 10. 31. ~ 2022. 11. 04.
- 개발환경 : HTML, CSS, JavaScript
- 웹사이트 구조:
	- 1 page 사이트 (한 화면에서 앵커로 이동하는 사이트)
    - 반응형 웹으로 동작, 자바스크립트를 사용한 동적인 효과가 2개 이상 포함
